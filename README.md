# React-WheatherApp
 
### Technologies Used :
  * React-JS
  * React Hooks
  * Netlify.com (for hosting)
  
### Deployment - Netlify
    https://cozy-stardust-702fe3.netlify.app
  
### How to Run Locally :
      
1. Install all dependencies
      
      `npm install`
      
2. Start Local Server
      
      `npm start`
